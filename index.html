Selected Works

English News Article De-noise

The American advertising company provides this topic. They wants to extract the main news content of English webpage news and provide 11k news links for use. First, use Selenium to dynamically render the web page and then crawl the HTML page. Use HTML2Text to extract all node text (including web pages template, advertisement, text), use Heuristic method to automatically label training data, process the webpage into a sequence of sentences, use Natural Language Processing, Sequence Labeling to train a deep learning model to deal with this problem, and predict that each sentence in the webpage belongs to the title, Abstract, text or other, to achieve the effect of extracting text.

Short URL

Server monitor bot

Due to the unstable power supply in the laboratory, the main computing server often automatically restarts or powers off, causing data broken or experiment interruption. Therefore, the development of Python combined with Slack bot api is used to monitor the status of each server at any time. Once an abnormality occurs, a message will be automatically sent to Slack groups. Note: This bot api is also used to notify the user when the program has completed execution.

Pattern Alignment

Develop and retrieve activity information from various webpages. Since activity information often appears in the same page as repeated blocks, our crawler system is used as the pre-processing module for webpage data, and train a Gradient Boosting Machine classifier to determine the appropriate encoding for the webpage, then encode the page node data to build a Prefix tree, extract the Longest repeat pattern in the tree, so that we cound find the blocks of the event information, and then use the Multi String Alignment method to align each event information.

Wrapper Verification

Use Tensorflow to train Sequence Labeling model to learn our crawler system (Unsupervised Web Data ETL) output, the original system can align multiple crawled web pages, use multiple rules to align the same attribute data and output Table For other purposes, each column in the original system Table is used as the label of the data, and when new webpage of the same domain crawled, it can be verified in the deep learning method, which can achieve higher efficiency than the original system and keep good results.
